# Tic-Tac-Toe Game

A responsive **Tic-Tac-Toe** web application built using **HTML**, **CSS**, and **JavaScript**. This project offers a fun and interactive two-player gaming experience with a clean user interface and game logic to detect wins, draws, and replay the game.

## 🌐 Live Demo

**🔗 Website:** https://tictac-toe-game-v2.netlify.app/

## 🚀 Features

* Two-player gameplay (Player X vs Player O)
* Interactive 3×3 game board
* Automatic winner detection
* Draw detection
* Reset and New Game functionality
* Responsive design for desktop and mobile devices
* Clean and intuitive user interface
* Smooth gameplay experience

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## 💻 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/DarshanPawar07/Project-06-Tic-Tac-Toe.git
```

2. Navigate to the project folder.

3. Open `index.html` in your preferred web browser.

4. Start playing by taking turns as **Player X** and **Player O**.
